QUESTION 30 · TEST AUTOMATION

How do you integrate accessibility testing into your automation?

Bạn tích hợp accessibility testing vào automation như thế nào?

Sample Answer (English)

I integrate accessibility testing by adding automated a11y scans directly into my existing test suite and CI pipeline. Using axe-core with Playwright, I inject the axe engine into a page after it loads and assert that there are zero violations against the WCAG guidelines, usually WCAG 2.1 AA. These scans catch issues like missing ARIA attributes, poor color contrast, and missing alt text automatically on every pull request. I also add keyboard navigation checks, verifying that I can tab through interactive elements and that focus order makes sense. That said, I always remind the team that automated a11y testing only catches around thirty to fifty percent of real accessibility issues, so we still rely on manual testing with screen readers like NVDA or VoiceOver for full coverage.

🇻🇳 Bản dịch tiếng Việt

Tôi tích hợp accessibility testing bằng cách thêm các bản quét a11y tự động thẳng vào bộ test hiện có và vào CI pipeline. Dùng axe-core kết hợp với Playwright, tôi inject engine axe vào trang sau khi trang tải xong và assert rằng không có vi phạm nào (zero violations) so với các hướng dẫn WCAG, thường là WCAG 2.1 AA. Các bản quét này tự động phát hiện những vấn đề như thiếu thuộc tính ARIA, độ tương phản màu kém, và thiếu alt text trên mỗi pull request. Tôi cũng thêm các kiểm tra điều hướng bàn phím, xác minh rằng tôi có thể dùng phím Tab để di chuyển qua các phần tử tương tác và thứ tự focus hợp lý. Tuy vậy, tôi luôn nhắc nhóm rằng automated a11y testing chỉ phát hiện được khoảng ba mươi đến năm mươi phần trăm các vấn đề accessibility thực tế, nên chúng tôi vẫn dựa vào manual testing với screen reader như NVDA hoặc VoiceOver để đảm bảo độ bao phủ đầy đủ.

💻 Ví dụ minh họa (Playwright + @axe-core/playwright)

// a11y.spec.js — quét accessibility tự động trong CI
const { test, expect } = require('@playwright/test');
const AxeBuilder = require('@axe-core/playwright').default;

test('trang chủ không có vi phạm accessibility (WCAG 2.1 AA)', async ({ page }) => {
  await page.goto('https://example.com');

  const results = await new AxeBuilder({ page })
    .withTags(['wcag2a', 'wcag2aa', 'wcag21aa'])
    .analyze();

  expect(results.violations).toEqual([]);
});

// Kiểm tra điều hướng bàn phím thủ công bổ sung
test('có thể tab qua menu điều hướng', async ({ page }) => {
  await page.goto('https://example.com');
  await page.keyboard.press('Tab');
  await expect(page.locator(':focus')).toHaveAttribute('role', 'navigation');
});
